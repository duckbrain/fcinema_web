Installation

Windows

After downloading fcinema the latest version, double-clik the executable file, the plugin will be added to it's folder and you can start watching the film.

Mac OS X

It's a little harder to install, but don't worry, it keeps being easy, and just after it leaves beta, you could just launch a .dmg. For now, go to applications folder and right-click Vlc.app. Press Show Package Content and navigate to /Contents/MacOS/share/lua/extensions/Plugins or create it. Then, you just have to copy the plugin file to that folder and start enjoying.

Linux

First of all, congratulations, the plugin is being developed over a Linux distribution so you will be the first on receive updates. That said, to install over a Linux distribution you have to copy the .lua file at /usr/lib/vlc/lua/extensions/

Using fcinema

Load the film as you would normally do. After that, once it is prepared, go to Vlc -> extensions and press fcinema, our software will start looking for the film in our database and it will show you the results. Select the sort of scenes you want to omit and start watching the film, you wouldn't ever notice that those scenes are being skipped
