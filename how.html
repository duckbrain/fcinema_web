Installation

Windows

After downloading fcinema's latest version, double-clik the executable file and... yep, that's all folks! It was easy, wasn't it?

Mac OS X

It's a little harder to install, but don't worry, it keeps being easy, and just after it leaves beta, you could just launch a .dmg. For now, go to applications folder and right-click Vlc.app. Press Show Package Content and navigate to /Contents/MacOS/share/lua/extensions/Plugins or create it. Then, you just have to copy the plugin file to that folder and start enjoying.

Linux

First of all, congratulations, the plugin is being developed over a Linux distribution so you will be the first on receive updates. That said, to install over a Linux distribution you have to copy the fcinema.lua file at /usr/lib/vlc/lua/extensions/ and thats all!

Using fcinema

Basic

Open

Load the film as you would normally do. Go to Vlc -> View and press fcinema.
That's all for you, our software will autorecognize the film and download the list on inapropiate content

Select

Select your personal criteria for inapropiate content. You can use fast filtering by category and severity or manually select specific scenes.

Enjoy

Click on Watch film and enjoy. You wouldn't ever notice that those scenes are being skipped

Advanced

Manually selecting the film

Sometimes autorecognition may fail. Introduce film title (currently only in english) and click "Select". If wrong film was selected go to Advanced -> Change Movie.

Manual calibration

fcinema automatically handle differences between film versions. However this is not always possible if we want to ensure a high quality calibration. A friendly scene is used to calibrate the film. You will be prompted to click when the scene begins. After checking the accuracy of the calibration click Okey and enjoy!

Changing the language

Go to Advanced, select your language and click Change language

Ninja

Adding new scenes

You can add new scenes by pressing "Q" during playback or by using the Scene editor
